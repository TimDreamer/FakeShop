# FakeShop

FakeShop is a project for training my own skills.

## Tech Stack

1. frontend
    - Vue, Vuex, Vue-Router
    - Sass
    - ApolloClient
2. backend
    - Typescript
    - GraphQL (ApolloServer)
3. TODO
    - Vue SSR

## Install

```
    $ yarn installAll
```

## Run

```
    $ yarn dev
```

### Note

-   default PORT / endpoint
    -   client: 8080
    -   server: 4000
    -   graphql: api/graphql
