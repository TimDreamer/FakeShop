# FakeShop

FakeShop is a project for training my own skills.

## Tech Stack

1. frontend (SSR)
    - Vue, Vuex, Vue-Router
    - Sass
    - ApolloClient
2. backend
    - Typescript
    - GraphQL (ApolloServer)

## Install

```
    $ yarn installAll
```

## Run

```
    $ yarn dev
```

### Note

-   default PORT / endpoint
    -   client: 8080
    -   server: 8081
    -   graphql: api/graphql
